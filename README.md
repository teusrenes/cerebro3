# cerebro3

This repository contains the starting point for the "cerebro3" project.

## Getting started

1. Clone the repository
2. Install dependencies (if any)
3. Start developing

## Development (recommended)

We recommend using a Python virtual environment for development to keep dependencies isolated.

Create the venv in the project root and activate it:

```bash
# create the venv
python3 -m venv .venv || python -m venv .venv

# activate (bash)
source .venv/bin/activate

# install dependencies (if any)
pip install -r requirements.txt
```

VS Code will usually auto-detect a `.venv` in the project root. We include a `.vscode/settings.json` that points the interpreter to `${workspaceFolder}/.venv/bin/python`.
